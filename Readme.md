# Привет!

Это главный репозиторий, в нем будут собраны все необходимые материалы для прохождения курсов по модулям:

- MySQL
- PHP
- Vue
- Laravel
- ...



> Информация в этом файле будет регулярно обновляться, поэтому следите за актуальностью!
